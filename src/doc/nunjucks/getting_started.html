1. Add Aerodial to your project

First, download the latest Aerodial and import it to your web page.

<!-- In the head tag -->
<link rel="aerodial-{{ config.version }}.min.css">

...

<!-- In the body tag -->
<script src="aerodial-{{ config.version }}.min.js"></script>

2. Create a pane

Create an instance of Aerodial using new Aerodial().

<script>
var aero = new Aerodial();
</script>

3. Ready to go!

Add components to fine-tune a property, monitoring a property, etc. See the reference on the left side for more detailed information.

// ...

var aero = new Aerodial();
aero.addControl(target, 'color');
aero.addControl(target, 'speed');

// ...
