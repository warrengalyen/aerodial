seasoner.js provides a text field for a string property.

var params = {
  name: 'cocopon',
};
var ss = Seasoner();
ss.add(params, 'name');

List

If you want to choose a value from list, use list option.

You can specify a string list:

var ss = Seasoner();
ss.add(target, 'direction').list([
  'left',
  'up',
  'right'
  'down'
]);

And also an object that has string values:

var ss = Seasoner();
ss.add(target, 'vimKey').list([
  left:  'H',
  up:    'K',
  right: 'L',
  down:  'J'
]);
