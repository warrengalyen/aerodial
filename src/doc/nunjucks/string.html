Aerodial provides a text field for a string property.

var params = {
  name: 'mechanika',
};
var aero = new Aerodial();
aero.add(params, 'name');

Selector

If you want to choose a value from list, pass a string list as values option:

var aero = new Aerodial();
aero.selector(target, 'direction', {
  values: [
    'left',
    'up',
    'right'
    'down'
  ]
});

An object that has string values is also supported:

var aero = new Aerodial();
aero.add(target, 'vimKey').list([
  left:  'H',
  up:    'K',
  right: 'L',
  down:  'J'
]);
