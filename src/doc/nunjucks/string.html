aerodial.js provides a text field for a string property.

var params = {
  name: 'cocopon',
};
var aero = Aerodial();
aero.add(params, 'name');

List

If you want to choose a value from the list, pass a string list as list option:

var aero = Aerodial();
aero.add(target, 'direction').list([
  'left',
  'up',
  'right'
  'down'
]);

An object that has string values is also supported:

var aero = Aerodial();
aero.add(target, 'vimKey').list([
  left:  'H',
  up:    'K',
  right: 'L',
  down:  'J'
]);
