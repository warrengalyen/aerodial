You can use on() of specific property to listen to its changes.

'change' event will be called continuously when the user controls a slider. 

var ss = Aerodial();
ss.add(target, 'speed', {min: 0, max: 100}).on(
  'change',
  function(value) {
    console.log(value);
  }
);

Global event

If you want to listen to changes of all properties, use on() of the root object.

var ss = Aerodial();
ss.add(target, 'power', {min: 0, max: 100});
ss.add(target, 'size', {min: 0, max: 100});
ss.on(
  'change',
  function(value, property) {
    console.log(property.getLabel());
    console.log(value);
  }
);

See Property for the specification of Property class.
