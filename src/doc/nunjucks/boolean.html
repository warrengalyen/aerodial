addControl() provides a checkbox for a boolean property.

var params = {
  debug: true
};
var aero = new Aerodial();
aero.addControl(params, 'debug');

Selector

You can use a named list for boolean value. Use addSelector() and pass an array of two string values (true, false) as values option.

var aero = new Aerodial();
aero.addSelector(target, 'debug', {
  values: ['Enabled', 'Disabled']
});
