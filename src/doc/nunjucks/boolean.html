Aerodial provides a checkbox field for a boolean property by default.

var params = {
  debug: true
};
var aero = Aerodial();
aero.add(params, 'debug');

List

You can use a named list for boolean value. Pass an array of two string values (true, false) to list option.

var aero = Aerodial();
aero.add(target, 'debug', {
  list: ['Enabled', 'Disabled']
});
