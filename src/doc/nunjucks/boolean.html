Aerodial provides a checkbox field for a boolean property by default.

var params = {
  debug: true
};
var aero = new Aerodial();
aero.checkbox(params, 'debug');

Selector

You can use a named list for boolean value. Pass an array of two string values (true, false) to values option to selector().

var aero = new Aerodial();
aero.selector(target, 'debug', {
  values: ['Enabled', 'Disabled']
});
