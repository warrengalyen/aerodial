Aerodial provides a text field for a number property by default.

var params = {
  speed: 0.5
};
var aero = new Aerodial();
aero.addText(params, 'speed');

Range

Use min or max option to specify minimum/maximum value. 
var aero = new Aerodial();
aero.addText(target, 'speed', {min: 0});

If you want to use a slider control, use addSlider() and specify both minimum and maximum value.

var aero = new Aerodial();
aero.addSlider(target, 'speed', {min: 0, max: 100});

Stepping

You can also specify a step of changing value by step option.

var aero = new Aerodial();
aero.addSlider(target, 'particleCount', {
  min: 0, max: 100, step: 10
});

Selector

If you want to choose a value from named values, pass an object that has number values as values option to addSelector().

var aero = new Aerodial();
aero.addSelector(target, 'direction', {
  values: {
    none: 0,
    horizontal: 1,
    vertical: 2,
    both: 3
  }
});
