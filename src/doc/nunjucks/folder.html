Use addFolder() to add a collapsible folder. It returns a instance of added folder, so you can add all types of property by using its add() and monitor() just like other properties.

var ss = Aerodial();

var folder1 = ss.addFolder('Interaction');
folder1.add(params, 'power');
folder1.add(params, 'range');

var folder2 = ss.addFolder('Appearance');
folder2.add(params, 'color')
folder2.add(params, 'size');

To close a folder, use close().

var ss = Aerodial();

var folder1 = ss.addFolder('Interaction');
folder1.add(params, 'power');
folder1.add(params, 'range');
folder1.close();

var folder2 = ss.addFolder('Appearance');
folder2.add(params, 'color')
folder2.add(target, 'size');
