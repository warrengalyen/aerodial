Aerodial has a feature for monitoring a property change. Use addMonitor() to monitor it.

var aero = new Aerodial();
aero.addMonitor(target, 'time');    // String
aero.addMonitor(target, 'active');  // Boolean
aero.addMonitor(target, 'color');   // Color
aero.addMonitor(target, 'wave');    // Number

Graph

addGraph() option provides a graph component to watch a number property.

var aero = new Aerodial();
aero.addGraph(target, 'wave', {
  graph: true,
  min: -5.0,
  max: +5.0,
  count: 200
});

min is a minimum value of the graph, max is a maximum value, and count is data count to plot.

Logger

To log a single-line string property, use addLogger.

var aero = new Aerodial();
aero.addLogger(target, 'time', {
  count: 10
});

count option is a count of data to hold.

Multi-line text

To monitoring a multi-line string property, enable multiline option.

var aero = new Aerodial();
aero.addMonitor(target, 'message', {
  multiline: true
});

Changing monitoring interval

You can change an interval for monitoring by interval option, in the order of milliseconds. The default value is 100ms.

var aero = new Aerodial();
aero.addMonitor(target, 'wave', {
  interval: 500,
  label: 'wave (0.5s)'
});
aero.addMonitor(target, 'wave', {
  interval: 1000,
  label: 'wave (1.0s)'
});
