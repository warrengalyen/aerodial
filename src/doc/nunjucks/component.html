Folder

Use addFolder() to add a foldable folder. It returns a instance of added folder, so you can add all types of property just like other properties.

var aero = new Aerodial();

var folder1 = aero.addFolder('Interaction');
folder1.text(params, 'power');
folder1.text(params, 'range');

var folder2 = aero.addFolder('Appearance');
folder2.text(params, 'color')
folder2.text(params, 'size');

To close a folder, use close().

var aero = new Aerodial();

var folder1 = aero.addFolder('Interaction');
folder1.text(params, 'power');
folder1.text(params, 'range');
folder1.close();

var folder2 = aero.addFolder('Appearance');
folder2.text(params, 'color')

Button

Use addButton() to add a button. You can listen 'click' event by on() to handle click.

var aero = new Aerodial();
aero.addButton('Fire').on(
  'click',
  function() {
    console.log('clicked');
  }
);

Separator

You can add a separator by addSeparator() to separate each component.

var aero = new Aerodial();
aero.addButton('Action');
aero.addSeparator();
aero.addButton('Import');
aero.addButton('Export');
