Changing a label of a property

Property name will be used as a label by default. Use label option to change it.

var aero = Aerodial();
aero.add(target, 'initSpeed', {
  label: 'Initial speed',
  min: 0.0,
  max: 1.0
});

Placing a aero in a specified element

You can specify a container element by container option.

var aero = Aerodial({
  container: document.getElementById('someElementId')
});

Disabling fold

Set foldable option to false.

var aero = Aerodial({
  foldable: false
});

Refreshing property views manually

Aerodial doesn't watch a property change automatically, so you should refresh manually if you change a property without Aerodial. Call refresh() to refresh all property views.

var aero = Aerodial();

// (changed a property without Aerodial...)

aero.refresh();
