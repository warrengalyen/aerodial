Aerodial

Import/Export

You can export all properties to JSON by getJson().

var aeroFrom = new Aerodial();
// ...
var json = aeroFrom.getJson();
console.log(json);

To import an exported JSON, pass it to setJson().

var aeroTo = new Aerodial();
// ...
aeroTo.setJson(json);

Changing a key of JSON variable

By default, property name will be used as a key of JSON object. You can change it by id option.

var aero = new Aerodial();
aero.addControl(target1, 'speed', {
  id: 'target1_speed'
});
aero.addControl(target2, 'speed', {
  id: 'target2_speed'
});
