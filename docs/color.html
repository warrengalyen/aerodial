Aerodial

Color

addPalette() provides a text field to edit a color property.

var params = {
  keyColor: '#00ff00'
};
var aero = new Aerodial({
  container: document.getElementById('colorExample')
});
aero.addPalette(params, 'keyColor');

A color picker is not provided yet, but I plan to implement it in the near future.
